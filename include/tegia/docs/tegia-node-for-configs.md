# Tegia Node: минимум контекста для конфигураций

`tegia-node` — это runtime-движок на C++ для акторов: он загружает типы акторов, создает инстансы, доставляет сообщения и выполняет actions асинхронно.

## Что дает runtime

- загрузка actor libraries (`*.so`) и регистрация actor types;
- маршрутизация сообщений к action акторов;
- worker pool и базовый lifecycle выполнения;
- инфраструктурные сервисы runtime (контекст пользователя, роли, dictionaries, DB providers).

## Что должно быть в конфигурационном репозитории (`*-conf`)

- описание включаемых actor types и параметров их запуска;
- конфигурация подключений (БД и внешние ресурсы);
- `patterns` и init-сообщения для старта;
- миграции и прикладные данные домена.

## Граница ответственности

- `tegia-node` не содержит бизнес-логику конкретного домена.
- Бизнес-логика, миграции и доменные параметры живут в `*-conf` репозиториях.
- Внутренняя архитектура runtime документируется в `/docs`, а SDK-контракты для конфигураций — в `/include/tegia/docs`.
